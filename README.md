# Proyecto Desarrollo Web – Coderhouse

Este proyecto fue realizado como parte del curso de **Desarrollo Web en Coderhouse**.  
Consiste en la construcción de un sitio web estático aplicando buenas prácticas de HTML, CSS y JavaScript.

## Tecnologías utilizadas
- **HTML5**: estructura semántica del sitio
- **CSS3**: estilos, diseño responsivo y maquetación
- **JavaScript**: interactividad básica en el frontend

## Funcionalidades
- Páginas estáticas con estructura clara
- Estilos personalizados y diseño adaptable
- Scripts simples para mejorar la experiencia del usuario

## Objetivo académico
- Practicar fundamentos de desarrollo web
- Aplicar conceptos de maquetación y diseño responsivo
- Integrar interactividad básica con JavaScript

## Cómo visualizar
1. Clonar el repositorio:
   git clone https://github.com/guadafestinese96/widistore.page.git
2. Abrir el archivo index.html en el navegador.
